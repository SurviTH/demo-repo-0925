## SubHeader - Commands Lesson

git status -> Show All file create update or delete

git add . -> Untrack to Track All file

git add [filename] -> Single File

git commit -m "Title" -m "Details" (Save Locally)

git push -> Update To Public (Online)

## Git Command Terminal.

git branch -> Check Current and Contain Branch

git checkout -> Switch Between Branch

git checkout -b -> Create A New Branch

## New Command Terminal

git diff [branch] -> Check differ thing contain between branch

git merge [branch] -> Merge new content to main branch

git branch -d [branch] -> Delete Branch

## New Commend Terminal

git commit -am "" //This command line for modify the file only.

## Undoing git

    - git reset [filename / HEAD~[point{1,2,3, ...}]] reset state when commit and push error
    - git log for check commit history

* Spacebar on keyboard when [git log] to skip to the end history display.
